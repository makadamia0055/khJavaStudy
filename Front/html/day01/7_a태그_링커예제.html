		이동하는 기능
		다운로드하는 기능

		a태그 자체로는 별 기능이 없음. 속성 값이 기능을 부여.
		링커기능 : 외부이동
		앵커기능 : 내부이동
  1. href : 링크한 페이지의 id값이나 주소 지정
  2. target : 링크 페이지가 표시될 위치(새 창, 현재 창) 지정
  3. download : 링크한 페이지를 표시하지 않고 다운로드 하는 것.
  4. rel : 현재 페이지와의 관계 지정
  5. hreflang : 링크한 페이지의 언어 지정
  6. type : 페이지의 파일 유형 지정
