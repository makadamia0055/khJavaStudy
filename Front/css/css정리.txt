스타일 적용 방법
1. 내부 스타일 시트(html 파일에 적용)
1-1. style 태그에 작성 -일반적으로 header태그에 작성
<style>
선택자 { 속성명 : 속성값 ; }
선택자 { 속성명 : 속성값 ; }
</style>

1-2. 각 태그의 style 속성으로 작성
<div style="속성명: 값; 속성명:값; 속성명:값"> </div>

2. 외부 스타일 시트(만들어진 css 파일을 연결)
link 태그를 이용하여 연결



css 선택자

전체 선택자 : *
태그 선택자 : 태그()
아이디 선택자 : #아이디명
클래스 선택자 : .클래스명

em: 부모 기준 상대 넓이
rem : 최상위 부모(root)기준 상대넓이

px:픽셀

vh: 브라우저 높이를 100으로 나눈 값
vw: 브라우저 가로 길이를 100으로 나눈 값
deg: 각도


글꼴 속성
font-family
폰트의 글꼴을 설정해주는 속성, 글꼴 1이 없으면 글꼴2,... 이렇게 순차적으로, 다 없으면 브라우저 기본 글꼴
font-size 글자 사이즈 단위 em, px, pt
font-weight 글자 굵기를 조절하는 속성

text-decoration 밑줄, 윗줄, 취소선
text-transform 대소문자 변환
text-shadow 글꼴에 그림자
