Thread 쓰레드

지금은 개념정도만 알아두기

프로세스 
	실행중인 하나의 프로그램
		하나의 프로그램이 다중 프로세스 만들기도

멀티 태스킹
	두가지 이상 작업을 동시에 처리하는 것

멀티 프로세스 
	독립적으로 프로그램들을 실행하고 여러가지 작업 처리

멀티 스레드
	한 개의 프로그램을 실행하고 내부적으로 여러 가지 작업 처리
	한 프로세스 내에 여러가지 스레드가 돌아감



메인 스레드
	모든 자바 프로그램의 기초
	main() 메소드의 첫 코드부터 아래로 순차적으로 실행
	실행 종료 조건
		마지막 코드 실행
		return문을 만남

	main 스레드는 작업스레드들을 만들어 병렬로 코드들 실행

싱글 스레드 : 메인 스레드 종료시 프로세스 종료
멀티 스레드 : 모든 스레드 종료되어야 프로세스 종료

작업 스레드 생성방법

	1. Thread 클래스로부터 직접 생성
		Runnable을 매개값으로 갖는 생성자 호출

	2. Thread 상속 후 run 메소드 오버라이딩해서 사용


동시성 : 싱글 코어에서는 멀티 스레드를 번갈아가면서 실행

병렬성 : 멀티 코어에서는 개별 스레드를 각각의 코어에서 동시에 진행

스레드 스케줄링
	스레드 의 개수가 코어 수 보다 많을 경우 실행 대기상태에 놓인 스레드들의 실행 순서를 결정해주는 것

java의 스레드 스케줄링
	우선순위 방식과 순환할당방식

	우선순위 방식 (코드로 제어 가능)
	우선 순위가 높은 스레드가 실행상태를 더 많이 가지도록 스케쥴링 

	순환 할당 방식
