7. update.jsp
 - 서버에서 보낸 게시글 정보를 이용하여 화면에 구성
 - form 태그 이용
 - action : /board/update
 - method : post
 - encType : multipart/form-data

- 첨부파일을 a태그로 작성
  -x버튼을 추가해서 x버튼을 클릭시 input태그로 변화 후, 
  -input 태그 hidden으로 삭제한 첨부파일 번호를 관리
  -name : fileNums

- 게시판 리스트를 이용하여 게시판을 구성

8. 컨트롤러
	- 메소드 추가
	-value : /board/update/게시글번호
	-method : post
	-viewName : /common/message
	- 로그인한 사용자 확인

	- 삭제할 첨부파일 번호들을 매개변수로 추가
	- 서비스에게 게시글정보, 사용자 정보, 첨부파일 배열, 첨부파일 번호 배열을 주고 게시글을 수정하라고 요청
	- 성공하면 msg에 "게시글 수정 성공!" 실패시 "게시글 수정 실패!"를 저장하고 화면에 전송

9. 서비스/임플
- 메소드 추가
- 게시글 null 체크
- 사용자 null체크

- 가져온 게시글이 null이면 실패를 리턴
- 있으면 게시글 작성자와 사용자 아이디가 다르면 실패를 리턴

- 반복문을 이용하여 삭제할 게시글 본호 배열을 첨부파일 리스트로 변환
	-첨부파일 번호를 꺼냄
	-DAO에게 첨부파일 번호를 주면서 첨부파일 정보를 가져오라고 요청
