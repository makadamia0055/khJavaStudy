-상품은 대분류, 중분류로 구성
-모든 상품은 대분류, 중분류 순으로 분류가 됨(모든 제품은 분류가 2가지 : 대분류, 중분류를 포함)
-모든 제품은 회원만 구매 가능
-제품을 장바구니에 수량을 선택하여 담을 수 있음
-배송지는 주소록에 등록된 배송지를 선택함. 직접 선택은 불가\
-제품은 옵션을 한가지만 가질 수 있고, 옵션 종류는 다양
-제품은 구매, 배송준비, 배송중, 배송완료, 구매확정, 환불요청, 반품요청, 반품완료, 환불완료를 할 수 있음
-공지사항과 Qna가 있음
-제품 문의를 할 수 있음.
-결제 카드만 가능(카카오 페이 가능)
- 할인, 쿠폰 X
-모든 제품은 구매금액의 10%를 적립하고, 포인트는 제품 구매시 사용할 수 있음.
-장바구니에 있는 여러 제품을 한번에 구매할 수 있음
-부분환불 , 반품은 X
-문의는 공개/비공개 선택가능
-한문의당 답변은 1개만 가능
-배송비는 무료

1. 데이터 정의어(DDL : Data Definition language)


mysql에서는 스키마와 데이터베이스를 (본질적으로는 다르지만) 같은 것으로 봄.

create : 데이터베이스, 테이블 등을 생성

데이터베이스 생성
	CREATE SCHEMA 스키마명;
	=CREATE DATABASE 데이터베이스명;

Drop : 데이터베이스 지우기
	Drop Schema 스키마명;

테이블 생성
Create table 테이블명(컬럼1 데이터타입[DEFAULT 형식], 컬럼명2 데이터타입 [DEFAULT 형식]
